# 📊 Project overview:

This project provides a comprehensive overview for the business requirement of the two distinguished Hotels in the world as per the need of the client. It consolidates key performance requirements and catalog metrics into an interactive, visually engaging format for strategic analysis.

# ❓KPI requirements:

* Total booking placed in Resort hotel and City hotel
* Average room occupancy rate per day, week and month (Helps measure how effectively rooms are being utilized).

## 💡Business requirements for the booking demand in both hotels:

This dashboard gives a clear overview of data analysts as per the business requirement:

* How many days in total advance customers book rooms varies on both Hotels?

* Need insight on booking reliability and revenue risk.

* What is the Average revenue earned of the Hotels?

* What is the percentage of bookings come from each channel? 

* What is the customer satisfaction score or review rating for Hotels?

* To know more about on repeat guest / loyalty return rate?

* What is the no-show rate for confirmed bookings?

# 🛠️ Tool and technologies used:

● Power BI – Dashboard creation & visualization

● Power Query – Data cleaning, data transformation
